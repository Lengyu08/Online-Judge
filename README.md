### 环境准备

```bash
create-react-app oj-fe
npm i redux react-redux @reduxjs/toolkit # react 树
npm i react-router-dom # 路由组件
npm i bootstrap # 响应式布局
npm install react-markdown # markdown 解析器
npm install remark-gfm # markdown 辅助解析
npm i markdown-navbar --save # markdown 导航栏
npm install github-markdown-css # mardown css
```
